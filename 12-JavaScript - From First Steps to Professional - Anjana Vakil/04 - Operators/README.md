# Operators

**Arithmetic Operators**
	-	`+` add
	-	`-` subtract
	-	`*` multiply
	-	`/` divide
	-	`**` Exponentiation
	-	`%` Remainder

**Order of operations**
4 + 1 * 2 * 4 + 2
same as in math, use ( ) to group
	-	( ( 4 + 1 ) * 2 * 4 ) + 2


**Comparison Operators**
	- `>` greater than
	- `<` less than
	- `>=` greater than or equal to
	- `<=` less than or equal to

**Equality Operators**
- `===`  `==` equals
- `!==`  `!=` does not equal

`1 === "1"`  -> `false`
`1 == "1"` -> `true`

You should almost always use the `strict` version
